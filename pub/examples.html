Tcal

Represent talents' public appearances nicely with this Javascript calendar library.

Description

The library is simple, like every other calendar library you might have seen, but geared towards representing talents' public appearances with the various options.

There is a very self-explanatory calendar layout on the left, and on the right is an "event panel", where it either shows the events on the day the user selected (try clicking on the days!), or the upcoming 5 appearances at the time being, including today.

Usage

Find this calendar nice and want to use it for other general purposes? Sure! We have designed this library such it accommodates these types of usages. Simply check out the API function for enabling this flexibility.

Examples


Without all the fancy stuff

Sometimes we don't need all that fancy, we only want to display everything, and that's it. This example demonstrates how to use the calendar without all the clickable events on the right, it just displays everything. 

                
                    const tc = new TalentCalendar(2021, 2022, 3, 2022, document.getElementById("first-example"));
                    tc.addEvents(events); // events: a list of events
                    tc.createCalendar();

With the fancy stuff

This one has everything applied. With the clickable events showing on the side.

                
                    const tc = new TalentCalendar(2021, 2022, 3, 2022, document.getElementById("first-example"));
                    tc.addEvents(events); // events: a list of events
                    tc.createCalendar();
                    tc.enableClickableEvents();

Colors? Set them as your liking!

You can also set your own color scheme! This example here changed the color-coding of the calendar.

                
                    const tc = new TalentCalendar(2021, 2022, 3, 2022, document.getElementById("first-example"));
                    tc.addEvents(events); // events: a list of events
                    tc.customizeColorCoding([
                      {
                        eventType: "stand-up",
                        color: "#f5ba83",
                      },
                      {
                        eventType: "tv-show",
                        color: "#8df5cf"
                      },
                      {
                        eventType: "quiz-show",
                        color: "#c9eec3"
                      },
                      {
                        eventType: "default",
                        color: "#f38bb7"
                      },
                      {
                        eventType: "theatre",
                        color: "#ce6fe5"
                      }
                    ]);
                    tc.createCalendar();
                    tc.enableClickableEvents();

Want to use it for general purposes?

Sure! Simply indicate it with the function forGeneralUsage()!

                
                    const tc = new TalentCalendar(2021, 2022, 3, 2022, document.getElementById("first-example"));
                    tc.addEvents(events); // events: a list of events
                    tc.forGeneralUsage();
                    tc.createCalendar();
                    tc.enableClickableEvents();

API

Below are all the API functions available in Tcal

Function Description
TalentCalendar(startYear, endYear, currMonth, currYear, parentElement) Constructor.
  • startYear: oldest year in the calendar
  • endYear: newest year in the calendar
  • currMonth: default display month
  • currYear: default display year
  • parentElement: HTML element that the calendar is to be inserted to
createCalendar() Creates a calendar instance on the element parentElement that is indicated when initializing the TalentCalendar instance
addEvents(events) Add the list of appearances events to the calendar view
enableClickableEvents(),
disableClickableEvents()		 Enable/Disable clickable side panel items.
customizeColorCoding(newColorCoding) Changes the color-coding of the calendar, newColorCodingis a list of ColorCoding objects. This should represent the full color scheme, including default colors.
forGeneralUsage() Gets rid of the disclaimer at the bottom of event pop-ups. Only call this when you want to use this library for general purposes.

Objects Explained

When using the library, you will encounter some JS objects that are necessary to using the library correctly.

            
                Event: // an event showed on the calendar
                {
                  eventName: "name of the event",
                  eventType: "type of event, used for color coding",
                  startTime: "Date object, start time of event",
                  endTime: "Date object, end time of event",
                  location: "location of appearance, can be TV channel or physical location",
                  officialWebsite: "url to official website of the event, nullable iff forGeneralUsage() is called",
                  description: "description of the event/appearance, nullable",
                  host: "host of show, nullable, list of person objects",
                  guests: "guests of show, nullable, list of person objects",
                  keyVisual: "key visual of event, nullable, url to an image",
                  ticketingInfo: "ticketing information, nullable, ticketing information object",
                  cast: "cast of the show/film/tv-drama, list of person objects",
                  recurringEvent: "boolean, true if this event is a recurring event, nullable",
                  recurrence: "recurrence information, nullable iff recurringEvent == false, recurrence object"
                }

                Person: // a person object, represents a person, used in host, guests, cast
                {
                  name: "name of person",
                  image: "url to image of person"
                }

                Ticketing Information: // a ticketing information object, represents ticketing information, used in ticketingInfo
                {
                  description: "a short summary of the ticketing information",
                  link: "url to the ticketing website"
                }

                Recurrence: // represents recurring event information, used in recurrence
                {
                  numberOfTimes: "integer, how many times this event will repeat",
                  repeat: "frequency of repeat. available options are 'daily', 'weekly', 'monthly'"
                }

                ColorCoding: // color coding of calendar
                {
                  eventType: "used as class name, put ‘default’ to change theme color",
                  color: "color code of above eventType",
                }

About

As someone who likes catching live streams of appearances on TV, the "media appearance" page on some talents' homepage is simply too unreadable. A calendar like this would a) greatly increase readability, b) way more convenient to find the show you are looking for, c) just look more organized while benefiting the users, us!
